   15.  Handling Different Functions in Javascript: Single Threaded Programming Language because JS has only one call stack to execute functions.

                1. Syncronous Functions: Executes line by line, each step blocks the next one until its completes.
                    - Blocking execution
                    - Runs in sequence
                    - Next statement wats for the current one to finish
                    - To execute these functions in JS engine uses the call stack
                    - eg: Array methods, string operations , localstorage access......

                2. Asyncronous Functions:  Allows code to run without blocking the main thread.
                    - They are essential for handling time-consuming tasks like API call, file loading, timers and DB operations.
                    -  Non blocking execution.
                    - eg: Fetching data form a server, Reading files, timers, DB operations.

                --- Ways to handle Asyncronous functions---

                        - Callbacks: A function passed as an argument to another function. Nesting  of callback functions leads to callback hell, hard to read and maintain.

                        - Promise: Represents a value that may be available now, later or never.
                            - Promise 3 states; pending / fullfilled/rejected
                            - Promise is a predefined class, First we need to create an object for the class.
                            - To resolve fullfilled cas use .then method of Promise
                            - To Resolve rejected case use .catch method of Promise

                        - Async-Await keywords : used to handle Asyncronous operations in a clean and readable way(Build on promise)
                            - async keyword : Declares an Asyncronous function, Always return a promise.
                            - await keyword : Pauses the execution until a promise resolves, Can be used only inside async functions, use try-catch block for reject case of promise.
